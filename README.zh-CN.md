# MyWindows10Tweak

Windows 10 Tweak AHK 脚本

**如有意见或建议请反馈。**

## 特性

* Win-M: 移动窗口
* Ctrl-H: 切换显示隐藏文件
* CapsLock: Ctrl; Shift-CapsLock: CapsLock
* 禁用 NumLock 键
* Win-Z: 最小化窗口
* 拖动时，方向键控制指针; 否则 Ctrl-Alt-方向键 控制
* 在任务栏上滚动滚轮: 调节系统音量
* 鼠标指针撞击开始按钮: 切换开始菜单（单显示器）
* 自动清除死进程的托盘图标
* 自动跳过修改扩展名的警告
* 复制图片或文本，Ctrl-V 保存为文件
* Win-`(反引号): 实时窗口预览 (最小化时暂停)
* GUI、托盘菜单以及设置
* 语言本地化 (编辑 locale.ini)
